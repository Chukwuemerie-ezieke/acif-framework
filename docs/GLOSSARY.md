# Glossary of Key Terms

| Term | Definition |
|---|---|
| **ACIC** | AI Content Integrity Committee — the governance body responsible for overseeing ACIF implementation within an institution |
| **ACIF** | AI-Generated Content Integrity Framework — this framework |
| **ACO** | AI Content Officer — the designated person responsible for day-to-day ACIF pipeline management |
| **AGC** | AI-Generated Content — any text, image, audio, or multimedia produced by an AI system in whole or in part |
| **Age Band** | A developmental grouping (1–6) that determines content appropriateness standards based on student age and cognitive development |
| **AIMS** | AI Management System — as defined by ISO/IEC 42001:2023, a structured system for managing AI throughout its lifecycle |
| **Content Integrity** | The state of curriculum content being accurate, age-appropriate, culturally safe, pedagogically aligned, and free from harmful information |
| **Content Registry** | A log of all AI-generated content that has been verified and approved for classroom use |
| **Cross-border Transfer** | The transfer of personal data from Nigeria to another country, regulated by the NDPA 2023 |
| **DPIA** | Data Protection Impact Assessment — required under NDPA for high-risk processing of personal data |
| **DPO** | Data Protection Officer — person responsible for monitoring NDPA compliance |
| **FACT Method** | Find-Assess-Cross-reference-Tag — the ACIF hallucination detection protocol |
| **Five-Gate Pipeline** | The sequential verification process: Pre-Delivery Screening, Factual Verification, Age-Appropriateness, Curriculum Alignment, Final Approval |
| **Flesch-Kincaid** | A readability test that estimates the US school grade level needed to understand a text |
| **Gate** | A verification checkpoint in the pipeline where specific standards must be met before content advances |
| **Hallucination** | A confident but factually incorrect, fabricated, or misleading statement produced by an AI model |
| **Hallucination Log** | An institutional record of all detected AI hallucinations, used for trend analysis and tool evaluation |
| **ISO/IEC 42001:2023** | The international standard for AI Management Systems, providing a framework for responsible AI governance |
| **Lexile** | A measure of text complexity used to match readers with appropriately challenging content |
| **LLM** | Large Language Model — AI systems like ChatGPT, Gemini, Claude that generate text based on training data |
| **NDPA** | Nigeria Data Protection Act 2023 — the principal data protection legislation in Nigeria |
| **NDPC** | Nigeria Data Protection Commission — the enforcement authority for the NDPA |
| **NERDC** | Nigerian Educational Research and Development Council — responsible for curriculum development in Nigeria |
| **RAG** | Retrieval-Augmented Generation — an AI technique that grounds responses in provided source documents |
| **Red Flag** | An indicator in AI-generated content that requires automatic risk tier escalation |
| **Risk Tier** | A classification (1–4) that determines how many verification gates content must pass through |
| **Tier Upgrade** | Moving content to a higher risk tier when red flags or verification concerns are discovered |
| **UNESCO** | United Nations Educational, Scientific and Cultural Organization |
| **Verification Gate** | See "Gate" |
| **WAEC** | West African Examinations Council — responsible for senior secondary certification examinations |
| **NECO** | National Examinations Council — a Nigerian examination body for secondary education |
