## Gate 1: Pre-Delivery Screening

**Reviewer:** _________________________ **Date:** _______________

- [ ] Content is labelled as AI-generated
- [ ] Risk Tier assigned and documented
- [ ] Age Band confirmed for target audience
- [ ] AI tool name and version recorded
- [ ] Generation prompt documented or summarised
- [ ] Red flags screened (specific dates/stats without citation, real people referenced, medical claims, post-2020 events)
- [ ] Readability score checked (Flesch-Kincaid grade level matches target band)

**Result:** [ ] PASS — proceed to Gate 2 [ ] FAIL — revise and re-screen

## Gate 2: Factual Verification (FACT Method)

**Reviewer:** _________________________ **Date:** _______________

- [ ] All factual claims identified in the content
- [ ] FACT method applied to each claim:
  - F: Claims found and listed
  - A: Risk assessed for each claim
  - C: Cross-referenced against 2+ authoritative sources
  - T: Each claim tagged (VERIFIED / MODIFIED / REMOVED)
- [ ] Zero unverified claims remain
- [ ] Sources documented for all verified claims
- [ ] Statistics checked against original source data
- [ ] Hallucination Log updated (if any hallucinations found)

## Gate 3: Age-Appropriateness Review

**Reviewer:** _________________________ **Date:** _______________

- [ ] Vocabulary complexity matches age-band specifications
- [ ] No prohibited themes for the target age band
- [ ] Cultural references are Nigerian or universally familiar
- [ ] Gender representation is balanced (no stereotyping)
- [ ] Religious/ethnic references are respectful and balanced
- [ ] Visuals (if any) comply with age-band guidelines
- [ ] Reading level confirmed (Flesch-Kincaid or Lexile)
- [ ] Sensitive topics handled per band-specific guidelines

**Result:** [ ] PASS — proceed to Gate 4 [ ] FAIL — revise and re-review

**Signature:** _________________________ **Date:** _______________

---

## Gate 4: Curriculum Alignment Check

**Reviewer:** _________________________ **Date:** _______________

- [ ] Content aligns with NERDC curriculum objectives for the specified level and subject
- [ ] Learning outcomes match curriculum expectations
- [ ] Assessment items (if present) match WAEC/NECO format where applicable
- [ ] Content does not introduce topics outside curriculum scope for the level
- [ ] Pedagogical approach supports student-centred methodology (per NERDC)
- [ ] Content supports relevant NERDC competencies (ICT literacy, critical thinking, creativity, citizenship)

**Result:** [ ] PASS — proceed to Gate 5 [ ] FAIL — revise and re-check
